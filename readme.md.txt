# David Clark — Principal AI Platform Engineer

A cyberpunk-themed personal portfolio website built with HTML, CSS, and vanilla JavaScript. Features a dark/light mode toggle, interactive project pages, an AI chatbot, and a working contact form that logs submissions to Google Drive.

## ✨ Features

- **Dark/Light Mode Toggle** — Persists user preference via localStorage
- **Circuit Board Background** — Animated canvas background with glowing nodes
- **Interactive Experience Timeline** — Collapsible cards with expand/collapse animations
- **Project Detail Pages** — Click any project to view full details with metrics and tech stack
- **Skills Section** — Icon badges with pill-shaped skill tags in a responsive grid
- **AI Chatbot** — Answers questions about David's experience, skills, and projects
- **Contact Form** — Validated form that submits to Google Drive via Google Apps Script
- **Floating Resume Button** — One-click PDF download
- **Fully Responsive** — Optimized for desktop, tablet, and mobile
